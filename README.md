# Travel Management System

An automated travel booking and passenger management desktop solution developed in Java using Object-Oriented Programming (OOP) principles.

## Features
- Designed structured passenger profiles, tour packages, and transport scheduling workflows.
- Dynamic dashboard interface for booking tracking and route selection.
- Robust inheritance models and custom data structures utilized to optimize routes data.
- Absolute file-handling streams implemented to store and retrieve active bookings data persistently.

## Tech Stack
- Language: Java
- Framework: Java Swing / AWT (GUI)
- IDE: NetBeans
- Concepts: Object-Oriented Programming (OOP), File Handling (I/O)
